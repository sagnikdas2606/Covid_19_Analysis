# Covid_19_Analysis
A Python-based COVID-19 data analysis project that processes Excel datasets, cleans and merges data, and generates insights through summary tables and visualizations such as trends, country comparisons, and growth analysis. Built using Pandas, NumPy, Matplotlib, and Seaborn.


## Objective
Analyze COVID-19 confirmed, death, and recovery trends across countries and continents.

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn

## Features
- Data Cleaning & Standardization
- Country & Continent Summary
- Monthly Growth Analysis
- Death & Recovery Rates
- Trend Visualizations
- Top Affected Countries

## How to Run
pip install -r requirements.txt
python main.py
